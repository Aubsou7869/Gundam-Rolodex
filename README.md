Gundam Rolodex

## Project Description
Gundam Rolodex is a Java console application that allows users to manage a collection of Gundam records using a SQLite database. The application demonstrates object-oriented programming concepts including inheritance, abstraction, interfaces, polymorphism, constructors, composition, and access specifiers. Users can add, view, update, and delete Gundam records through a menu-driven interface.

 Project Tasks
Task 1: Set up the development environment**
  - Install Java JDK
  - Configure VS Code/Eclipse
  - Add SQLite JDBC library to the project
  - Configure GitHub repository

Task 2: Design the application**
  - Plan the class structure
  - Design the database schema
  - Create the menu-driven user interface

Task 3: Develop object-oriented classes**
  - Create abstract Gundam base class
  - Create CombatGundam derived class
  - Create Manufacturer composition class
  - Create Displayable interface

Task 4: Implement database functionality**
  - Connect to SQLite database
  - Create database table
  - Implement CRUD operations

Task 5: Implement user interaction**
  - Add menu options for CRUD operations
  - Validate user input
  - Display formatted Gundam information

Task 6: Test the application**
  - Test database connection
  - Test CRUD functionality
  - Debug compile/runtime issues

Task 7: Document the project**
  - Add code documentation
  - Create README/project summary
  - Add inline comments explaining OOP concepts
